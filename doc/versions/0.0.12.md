## 0.0.12 - 2026-03-31

### Added

1. SSE 通道升级为统一入口：前端改连 `/api/events`，后端同时兼容旧路径 `/api/state-events`；并新增 `notify_tasks` 事件广播，让通知任务在多窗口间可以由后端主动触发前端刷新。
2. 首页新增可切换位置的停靠式任务栏：支持通过“位置”下拉框切换到上、下、左、右四个方位；任务栏样式改为“整条停靠”（而非单按钮），并按需求简化为不显示“导航栏”标题和导航按钮。
3. 主内容区增加自动避让：任务栏切换到任意方位后，页面会自动留出对应边距，避免内容被任务栏遮挡。
4. 修复顶部停靠时与右上角 Status 的遮挡问题：顶部任务栏改为贴顶显示，并在计算主内容上边距时同时考虑 Status 区域，避免出现顶部空隙异常和内容被覆盖。
5. 任务栏新增“设置”按钮：点击后弹出设置面板，面板内可调整任务栏位置并可手动关闭。
6. 任务栏新增“统计”按钮：点击后打开独立的“统计弹窗”；统计内容包含“单日最大DP获取”“单日最大DP消耗”，用于直观看到当天单次最大入账/扣减。
7. 统计弹窗新增历史维度：增加“历史单日DP最大增量/最大减量（含日期）”，按项目全历史记录计算，便于快速看到历史峰值日。
8. 任务栏新增“状态”按钮占位：预留后续状态功能入口，当前不触发任何业务逻辑。
9. 任务栏新增“物品”按钮占位：预留后续物品功能入口，当前不触发任何业务逻辑。
10. 任务栏新增“记录”按钮：点击后打开“记录弹窗”，并把首页历史记录整合到该弹窗中。
11. 任务栏新增“工具”按钮：点击后打开“工具弹窗”，并把首页功能列表整合到该弹窗中。
12. 任务栏按钮顺序调整为：状态、物品、记录、工具、统计、设置。
13. “状态”“物品”按钮改为可点击：点击后显示可爱提示“开发中……喵~”，并自动渐变消失。

### Fixed

1. 通知任务在“无待触发任务”时停止轮询：移除固定 15 秒轮询，改为“有任务才开启轮询、无任务立即停轮询”，显著减少大量 `GET /api/notify-tasks` 请求。
2. 页面从后台切回前台时，通知任务会自动补拉一次；即使个别场景下 SSE 短暂断线，也能尽快恢复任务列表同步。
3. 进程监控在 server 自动重启后会先恢复活跃会话快照，再处理 stop 事件，避免“程序实际运行过但 `session_seconds` 记为 0”的问题。
4. 修复前端刷新时可能出现通知弹窗的问题：自动触发的通知任务改为静默模式（不申请通知权限、不弹 alert），避免页面重连/刷新时打断使用。
5. 修复“状态/物品”开发中提示在刷新或焦点变化时可能误触发的问题：提示仅在真实用户点击下触发，避免页面重载时自动弹出。
6. 进一步收紧“开发中提示”触发条件：改为只响应 `pointerup` 左键/触控抬起事件，避免服务重启导致页面刷新后再次误触发。
7. 修复服务重启后前端被强制整页刷新并“自己弹到前面”的问题：SSE 重连成功后取消 `window.location.reload()`，改为静默补拉数据更新。
8. 修复 Git 提交保活在异常多实例并发时可能重复提交的问题：给 `github_empty_commit` 执行器增加了文件互斥锁，同一时刻只允许一个执行器进入提交流程，避免在未到间隔周期时出现短时间连续空提交。
9. 修复主服务重复启动导致保活子进程被多开的问题：`server.py` 启动时新增单实例锁（`data/server.lock`），如果检测到已有主服务在运行则直接取消本次启动，从源头避免同一台机器并发触发多次保活提交。

### Changed

1. “今日日报（简略）”改为“每日日报（简略）”：新增左右箭头切换日期（后端支持 `day_offset` 查询参数，前端默认显示当天并可查看前几天）。
2. 每日日报中“没有变更”的项目不再显示：DP 或 GP 若当天无变更则省略该行，若两者都无变更则显示“当天没有 DP/GP 变更”。
3. 每日日报删除“最后一次变更”这一行，减少无关信息干扰，聚焦当天汇总数据。
4. 任务栏按钮交互改为“再点一次关闭”：设置、记录、工具、统计按钮在对应面板已打开时，点击同一按钮会直接关闭该面板。
5. 任务栏弹窗交互进一步统一：当打开任一任务栏面板（设置/记录/工具/统计）时，会先自动关闭其他面板，确保同一时刻只显示一个面板。
6. 任务栏弹窗新增“点空白关闭”：点击任务栏和弹窗区域之外的页面空白时，会自动关闭当前已打开的任务栏弹窗。
7. 任务栏弹窗新增 `Esc` 快捷关闭：按下 `Esc` 键时，自动关闭当前已打开的任务栏弹窗。
8. 任务栏按钮新增“当前面板高亮”状态：当设置/记录/工具/统计面板打开时，对应按钮会显示高亮，方便快速识别当前焦点。
