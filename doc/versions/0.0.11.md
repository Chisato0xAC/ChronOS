## 0.0.11 - 2026-03-31

### Added

1. 通知系统升级为“时:分:秒”延迟任务模式：支持可配置延迟、到点时间预览、待触发任务列表，并将任务落盘到 `data/notify_tasks.json`，刷新页面后可恢复未到点任务。
2. 首页新增“今日日报（简略）”：按换日时间（默认 04:00）统计当天有效状态变更，并展示统计日期、变更条数、DP/GP 合计变化与最后一次变更时间。
3. 首页完成一轮轻量结构与样式升级：引入有限度 CSS、调整为两栏布局（左侧操作与历史，右侧状态与功能）、将“当前数值”并入右栏状态区，提高首屏信息密度与阅读动线。
4. 首页新增“功能列表”，并将通知、便签、悬浮窗改为可收纳折叠区域；历史记录区域增加默认高度上限并启用纵向滚动，减少页面被长内容拉伸的问题。

### Fixed

1. 通知按钮文案与行为同步：改为按输入的秒数发送，并在通知正文里显示实际延迟秒数，避免“按钮写可配置但实际固定 5 秒”的误解。
2. 通知按钮文案更新为“按上面的时分秒发送通知”，与实际交互保持一致。
3. 通知“时”输入框补充 `max=99`，与“分/秒”输入框保持接近的显示宽度，避免“时”框明显更宽。
4. 预计通知时间与任务剩余时间改为每秒刷新，修复“时间在走但预览静止”的问题。
