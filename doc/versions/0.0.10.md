## 0.0.10 - 2026-03-26

### Fixed

1. 服务器启动端口支持读取环境变量 PORT，默认仍为 8000，方便 Render 生产环境直接启动。
2. 前端改为单页展示，不再做上下翻页；历史区域固定展示最近 8 条记录。
3. 前端页面主区域改为左右 70/30 布局，并将右侧区域固定顶部对齐，避免“右侧面板飘在中间”。
4. 进程结束监听增加异常兜底：遇到 WMI“配额冲突”时会等待 5 秒自动重连，避免线程直接退出。
5. 新增悬浮窗脚本：无窗框、置顶、半透明，并在右上角提供关闭按钮。
6. 悬浮窗支持鼠标拖动（按住任意位置即可移动）。
7. 前端增加“打开悬浮窗”按钮，并新增 `/api/open-floating-window` 接口用于启动悬浮窗。
8. 右侧预留框改为“便签（自动保存）”：输入内容后会自动调用后端保存，刷新页面后会自动读取 `data/note.json` 里的便签内容。
9. 悬浮窗背景升级为浅色纵向渐变，并修复关闭按钮与拖动手势冲突问题。
10. 修复“通知功能”和“历史记录”之间出现大块空白的问题：将历史记录区域移动到左侧同一列，紧接在通知功能下方显示。
11. 在“当前数值”的 DP 右侧新增“当前周期 DP 变更”显示：正数显示绿色、负数显示红色，并按每天 4:00 为周期起点统计本周期累计变化。
