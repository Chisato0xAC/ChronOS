## 0.0.13 - 2026-04-03

### Changed

1. 同步托管子进程规则文档到当前实现：`doc/rules/ManagedChildren.md` 中的配置路径由 `tools/managed_children.json` 更正为 `config/managed_children.json`，并把 `process_watch` 示例脚本路径更正为 `src/monitor/process_watch.py`。
2. 在任务栏中补上“商店”占位按钮，并把它放到真正的正中间：7 个按钮时位于第 4 个位置；同时补齐同一组样式规则，让它的宽度、间距和其他任务栏按钮保持一致，点击效果与“状态”“物品”一致，统一显示“开发中”。
3. 前端自动刷新调整为“前台立即刷新，后台延后刷新”：当后端重启、前端重连成功时，如果页面正在前台就立刻整页刷新；如果页面在后台，就先记住这次刷新，等你自己切回这个标签页时再自动刷新。这样像新增任务栏按钮这类前端结构改动也能自动更新，同时尽量避免浏览器自己跳到前面。
