
## 已完成功能（明细）

- 第一次运行 `server.py` 时，如果 `data/state.json` 不存在，会自动创建默认文件（dp=0, gp=0）。
- 新增“只显示一个窗口”的用户启动方式：运行 `ChronOS.vbs` 会后台启动 `server.py`，并用 Edge 的 app 窗口打开页面；当用户关闭该窗口时，会自动关闭服务器进程。
- 运行日志放在 `logs/`（`logs/server.out.log`、`logs/server.err.log`）。
- Edge 的临时用户数据目录使用系统临时目录下的 `ChronOS_edge_profile`，并在退出时自动清理，避免和个人浏览器数据混在一起。
