DP Cycle Run Cost Rule (Draft)

- 一个周期内（每天）程序可以运行多次。
- 结算时间使用本地已有设置（凌晨 4 点重置），这里不新增 `settlement_hour` 配置。
- 同一周期内，第 N 次运行的倍率是 N。
- 单次消耗公式：`消耗 = 运行分钟数 × 倍率(N) × 基础单价`。
- 例子：
  - 第 1 次运行 X 分钟，消耗 `X × 基础单价`。
  - 第 2 次运行 Y 分钟，消耗 `2Y × 基础单价`。
- 周期总消耗 = 该周期内所有运行段消耗之和。
- 如果到周期结算点时程序仍在运行，触发 CHAOS 规则（细则待定）。
- CHAOS 规则先保留占位：`chaos_rule = TBD`。
- 说明：基础单价可能调整，计算时以当前配置/实现为准。
